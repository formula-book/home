  • Arithmetic progression
    • General form : a, a + d , a + 2d ..., where 'a' is the first term and 'd' is the general difference.
    • nth term : a + (n - 1)d
    • Summation of n terms ( Sn ) = n⁄2 . [2a + (n - 1)d]
    • Selection of terms:
      • 3 terms : (a - d), a, (a + d)
      • 4 terms : (a - 3d), (a - d), (a + d), (a + 3d)
      • 5 terms : (a - 2d), (a - d), a, (a + d), (a + 2d)
      • Odd : Common difference d, Even : Common difference 2d
    • If a, b, c are in Arithmetic progression, then 2b = a + c
